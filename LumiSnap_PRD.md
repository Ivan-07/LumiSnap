# LumiSnap 产品需求文档（PRD）

---

## 1. 产品背景和目标

**产品定位：**
LumiSnap 是一款基于多模态 AI 的智能拍立得相机应用，旨在通过"即拍即得"的方式，将图片内容与风格化文案智能联动，极大简化内容创作流程。

**要解决的问题：**
- 降低用户生成高质量图文内容的门槛，消除"不会写文案""流程繁琐"等痛点。
- 捕捉用户灵感瞬间，避免创作心流被多步操作打断。
- 满足内容创作者、电商卖家、日常分享者等多元场景下的高效表达需求。

**核心目标：**
- 实现"选场景-拍照-得文案"一体化极简体验。
- 支持多种风格模板，满足不同平台和场景的内容需求。
- 提供可扩展的接口和数据结构，便于后续自动化和智能工具集成。

## 2. 核心功能模块列表

| 功能模块 | 功能描述 | 使用场景 | 解决的问题 |
| :--- | :--- | :--- | :--- |
| 相机/拍摄界面 | 提供基础拍照功能，支持取景、对焦、拍摄。 | 用户需要捕捉当前场景或物品时 | 降低拍照门槛，保证图片输入质量 |
| 场景模板选择器 | 让用户选择不同文案风格模板（如小红书、咸鱼、朋友圈等） | 用户希望生成特定风格文案时 | 解决不会写Prompt、风格不统一的问题 |
| AI 生成引擎 | 根据图片和模板，自动生成匹配的文案 | 用户拍照后自动获得文案 | 解决文案创作难、效率低的问题 |
| 结果展示与编辑页 | 展示图片和AI生成文案，支持基础编辑和微调 | 用户查看、调整、确认最终内容 | 解决AI结果不完美、需个性化调整的问题 |
| 导出与分享 | 支持保存图片/文案、复制、分享到外部平台 | 用户需要将内容发布或保存时 | 解决内容流转、跨平台分享需求 |
| 首次用户引导 | 新用户首次进入时的操作指引 | 新用户首次使用产品 | 降低上手门槛，提升留存 |

## 3. 用户流程图（文字版）

1. 用户打开 LumiSnap，首次进入时触发新手引导。
2. 进入主界面，系统请求相机权限。
3. 用户在相机界面取景，滑动选择所需文案风格模板。
4. 用户点击拍照按钮，系统采集图片并记录所选模板。
5. AI 生成引擎处理图片和模板，生成风格化文案。
6. 跳转到结果展示页，用户查看图片和文案。
7. 用户可选择：
    1. 直接导出/分享内容
    2. 对文案进行微调后再导出/分享
8. 用户完成内容保存或分享，流程结束。

## 4. 每个功能的输入/输出

| 功能模块 | 输入 | 输出 |
| :--- | :--- | :--- |
| 相机/拍摄界面 | 用户操作（取景、拍照）、相机权限 | 图片文件（JPEG/PNG）、拍摄元数据 |
| 场景模板选择器 | 用户选择模板 | 模板ID/名称、模板参数 |
| AI 生成引擎 | 图片文件、模板ID/参数 | 生成的文案文本、置信度分数（可选） |
| 结果展示与编辑页 | 图片、文案文本 | 用户编辑后的文案、最终确认内容 |
| 导出与分享 | 用户操作（保存/复制/分享） | 本地图片、复制到剪贴板、分享API调用结果 |
| 首次用户引导 | 用户滑动/点击 | 完成引导标记（本地存储） |

## 5. 接口需求

| 功能模块 | 是否需要接口 | 接口用途 | 调用方式 | 数据结构要求 |
| :--- | :--- | :--- | :--- | :--- |
| 相机/拍摄界面 | 否 | - | - | - |
| 场景模板选择器 | 否（本地/内置） | - | - | - |
| AI 生成引擎 | 是 | 图片+模板参数上传，返回文案 | RESTful API / WebSocket | 输入：图片二进制流、模板ID；输出：文案文本、状态码 |
| 结果展示与编辑页 | 否 | - | - | - |
| 导出与分享 | 视平台而定 | 分享到第三方平台 | 分享API/系统分享 | 输入：图片/文本，输出：分享结果码 |
| 首次用户引导 | 否 | - | - | - |

## 6. 边界条件和异常处理

| 场景 | 可能的异常/边界情况 | 系统响应 |
| :--- | :--- | :--- |
| 相机权限 | 用户拒绝相机权限 | 弹窗提示"需开启相机权限才能使用"，引导用户到设置页 |
| 拍照失败 | 相机硬件异常、存储空间不足 | 提示"拍照失败，请重试"，并记录日志 |
| 模板未选择 | 用户未选择模板直接拍照 | 默认选中上次模板或第一个模板 |
| AI生成失败 | 网络异常、API超时、AI服务不可用 | 提示"文案生成失败，请重试"，可重试或保存图片待后续生成 |
| 文案不满意 | 用户多次点击"换一句"仍不满意 | 提供基础编辑框，允许手动修改 |
| 分享失败 | 第三方平台接口异常 | 提示"分享失败，请稍后重试"，允许用户保存到本地 |
| 首次引导中断 | 用户中途退出引导 | 下次进入时继续未完成的引导步骤 |
